# Dojo
Estrutura para Dojo's de PHP com autoload do Composer.

## Pré-requisitos
1. PHP na máquina

## Tutorial
1. Clone este repositório

```sh
$ git clone https://github.com/verzola/Dojo.git
```

2. Instale as dependências pelo Composer

```sh
$ php composer.phar install
```

3. Rode os testes!

```sh
$ bin/phpunit
```
